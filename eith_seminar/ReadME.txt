Модуль "main" содержит основную логику программы базы данных сотрудников
Модуль "user_select" содержит функции выбора уровня доступа (администратор или гость) и возможных операций с БД
Модуль "operations" содержит все функции операций
Модуль "file_work" содержит функции чтения и записи файла BD.json
